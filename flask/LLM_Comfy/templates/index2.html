Ollama Generate API

단일 프롬프트를 빠르게 실행합니다.

Ollama의 generate 엔드포인트를 Flask에서 스트리밍으로 연결한 테스트 페이지입니다. 모델과 프롬프트를 입력하면 응답이 실시간으로 출력됩니다.

Mode Single Prompt
Endpoint /api/generate
Streaming NDJSON Response
Default Model gemma4:e2b

Generate Request

간단한 질의, 문장 생성, 번역, 요약 같은 1회성 요청을 테스트할 때 사용합니다.

Ollama 응답을 스트리밍으로 수신하는 중입니다.

Output

모델 응답이 도착하는 즉시 아래 영역에 누적됩니다.

⌁
아직 응답이 없습니다. 왼쪽에서 프롬프트를 입력하고 Generate를 누르세요.
